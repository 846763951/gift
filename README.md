# ECD 2013 抽奖源代码

-----
## 项目启动

全局安装 Node 模块 anywhere, 然后在项目根目录跑 anywhere 命令

## 奖池数据

js/kalok.js: 抽奖程序主代码

data/thing.js: 奖品信息（奖品的 ID 一定要从 1 开始自增）

data/user.js: 参与抽奖人员信息

thing/xxx.png: 奖品图片

## 动画效果

animate/loading.gif: 抽奖中的 loading 动画

## 头像

rtx/xxx.png: 参与抽奖人员头像(头像名称需要和 user.js 中的 rtx 字段一一对应)
